# Config reference

- `association_mode`: `nearest` or `global_greedy`
- `motion_model`: `constant_velocity` or `damped_velocity`
- `gate_radius`: Euclidean gate in world units
- `process_noise`: prediction uncertainty factor
- `measurement_noise`: measurement trust factor
- `confirm_hits`: hits needed before a track becomes confirmed
- `max_misses`: misses allowed before deletion
- `camera_confidence_scale`: weighting factor for camera detections
- `radar_confidence_scale`: weighting factor for radar detections
